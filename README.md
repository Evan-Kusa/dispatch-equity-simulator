# Dispatch Equity Simulator

Dispatch Equity Simulator is a browser-based ethics simulation about autonomous last-mile delivery. It lets the user reallocate a fixed priority budget across speed, throughput, profit, and equity, then watch how those choices change wait times, profit, and neighborhood-level exclusion.

## Live site

Public URL: `https://evan-kusa.github.io/dispatch-equity-simulator/`

## Local use

This repo is intentionally minimal. Open `index.html` directly in a browser, or serve the folder locally with a simple static server.

## Repo contents

- `index.html`: the complete interactive simulator
- `README.md`: project overview and usage notes
